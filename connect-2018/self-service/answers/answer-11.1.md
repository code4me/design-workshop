## Answer to exercise 11.1


Replace the `{{brand}}` widget with the following HTML:

``` html
<div class="logo">
  <a href="/self-service">
    <img src="[URL from media library]">
  </a>
</div>
``` 

In the `CSS` field, add the following CSS:

``` css
.global-navigation {
  .logo {
    margin: 20px;
  }
  
  img {
    width: 100%;
  }
}
```
